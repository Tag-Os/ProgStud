# Задание 2

## Описание задания

Расставьте знаки операций "плюс", "минус", "умножение" и скобки
между числами "1 2 3 4 5" так, что бы получилось число "25".

## описание работы

$$ ((1 + 2) \times 3 - 4) \times 5 $$
