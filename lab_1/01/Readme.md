# Задание 1

## Описание задания

Есть значение радиуса круга,выведите на консоль значение прощади этого круга с точностю до 4-х знаков после запятой,а также лежит ли точка внутри круга или нет.

## описание работы

Вычисленна площадь по формуле: $$ S = \pi r^2 $$

А проверка положения точки с помощью формулы:  $$ d = \sqrt{x^2 + y^2} $$
