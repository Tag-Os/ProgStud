# Задание 6

## Описание задания

Есть список и словарь песен группы Depeche Mode со временем звучания,распечатайте общее время звучания некоторых из них.

## описание работы

Пример нахождения звучания:

```python
duration = round(violator_songs_list[3][1] + violator_songs_list[5][1] + violator_songs_list[-1][1],2)
```

аналогично находится время звучания и для другой группы
